### One day
1. 下载后面所需的assets资源 
2. 分析页面结构
   - 首页带四个tab: Feed Category Cart Profile
   - Feed子页面：搜索框，消息提醒
   - Category子页面： 子子页面
   - Cart子页面：购物车、订单
   - Profile 子页面： 我的订单、我的收藏、我的评价、地址管理、最近访问、账户信息、退出

   - 常见页面：
     - loading page
     - error page
     - notification not login page
     - login page
     - logout page
3. 创建所需要的constants(in utils)
